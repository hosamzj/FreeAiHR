# 功能增强路线图

## 当前状态（已具备）

- ✅ 多角色认证（admin / hr_manager / interviewer / candidate）
- ✅ 候选人管理（简历解析、AI 筛选、候选人池）
- ✅ 面试管理（排期、日历、评价、评分）
- ✅ Offer 管理（审批流、AI 建议）
- ✅ 系统设置（SSO、密码策略、基础配置）
- ✅ 用户管理（CRUD、角色分配）

---

## 阶段一：核心增强（建议优先）

### 1. 安全防护 🛡️

| 功能 | 优先级 | 说明 |
|------|--------|------|
| API 速率限制 | P0 | 登录/注册/API 防暴力破解，基于 IP 限制 |
| 操作日志完整化 | P1 | 当前审计日志忽略错误，应异步落盘 + 持久化存储 |
| 密码历史检查 | P1 | 修改密码时检查最近 N 次历史，防止复用 |
| 敏感数据加密 | P2 | 手机号、身份证号等字段数据库级加密 |

### 2. 通知系统 📢

| 功能 | 优先级 | 说明 |
|------|--------|------|
| 邮件通知 | P0 | 面试邀请、Offer 发送、密码重置邮件 |
| 站内消息 | P1 | 通知中心红点、未读计数、消息列表 |
| 日历同步 | P2 | 面试官 Google Calendar / Outlook 同步 |

### 3. AI 功能深化 🤖

| 功能 | 优先级 | 说明 |
|------|--------|------|
| 简历智能匹配 | P1 | 自动计算候选人与职位的匹配度并排序 |
| AI 面试助手 | P2 | 实时语音识别 + 自动记录面试笔记 |
| AI 薪酬建议 | P2 | 基于市场行情、候选人经验给出薪资范围 |

---

## 阶段二：体验优化

### 4. 数据可视化 📊

- 招聘漏斗转化图（简历 → 筛选 → 面试 → Offer → 入职）
- 部门招聘效率对比雷达图
- 面试官评分分布热力图
- 实时数据大屏（适用于 HR 管理层看板）

### 5. 移动端适配 📱

- 面试提醒推送（Web Push / PWA）
- 面试官移动端评价界面优化
- 候选人移动端面试签到

### 6. 导入导出 📁

- Excel 批量导入候选人
- 简历 PDF 批量解析
- 招聘报表导出（周报/月报）

---

## 阶段三：企业级功能

### 7. 多租户 / 多公司 🏢

- 同一系统支持多个公司独立使用
- 公司级别数据隔离
- 子账号权限管理

### 8. 集成第三方 🔗

- 企业微信 / 钉钉 / 飞书通知
- Boss 直聘 / 拉勾 / 猎聘简历导入 API
- 背景调查服务对接

### 9. 高级功能 ⚡

- 人才库激活（定期邮件触达储备候选人）
- 内推奖励系统（员工推荐追踪、奖金发放）
- 视频面试集成（Zoom / 腾讯会议 / 钉钉）

---

## 推荐实施顺序

```
Week 1-2:  邮件通知 + 站内消息 + 健康检查完善
Week 3-4:  数据可视化 + 导入导出
Week 5-6:  AI 简历匹配 + 面试助手
Week 7+:   移动端适配 + 第三方集成
```
